# Diabetes Management App

## Overview
A Streamlit-based diabetes management app that helps users track blood glucose levels, monitor diet, and visualize health data.

## Features
- User Authentication (Login/Signup)
- Role-based system (Patient, Doctor, Caregiver)
- Blood Glucose Tracking (Low, Normal, High, Critical)
- Meal Tracking (Sugar intake)
- Dashboard with graphs (Plotly)
- Simple and user-friendly UI

## Installation
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

## Run the App
streamlit run app.py

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## Goal
To provide an easy-to-use platform for managing diabetes effectively with data visualization and tracking.
